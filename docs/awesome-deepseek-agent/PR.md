## Summary

This PR adds an Orcana integration guide.

Orcana is a DeepSeek-native terminal coding agent runtime focused on long-running software engineering tasks. It supports planning gates, task packets, scoped tool use, patch transactions, evidence-based completion, and an interactive TUI.
